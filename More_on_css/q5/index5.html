Google Fonts Blog

Google Fonts is a library of over 1,000 free licensed font families, an interactive web directory for browsing the library, and APIs for conveniently using the fonts via CSS and Android. Google Fonts aims to make it quick and easy for everyone to use web fonts, including professional designers and developers. The library is open source, which means you can use them for any project. The fonts are also optimized for high performance, making sure your web pages load fast and look beautiful.
